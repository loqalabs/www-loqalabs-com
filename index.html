What is Loqa?

Loqa is a privacy-first voice assistant platform that runs entirely offline. It features hardware “pucks” for room-scale voice capture and a local hub that handles STT, LLM parsing, and command chaining.

Why Local-First?

  • No cloud dependencies
  • Data never leaves your network
  • Modular skills system
  • Runs on open-source models like Whisper and LLaMA
